
# Zabbix Template Markdown Generator
Генератор документации для шаблонов Zabbix с опциональным автоматическим переводом описаний (русский + оригинал).

---

## Описание

Скрипт предназначен для автоматического создания технической документации по шаблонам Zabbix (YAML-экспорт) в формате Markdown (`.md`).
**При использовании флага `--translate`** в документации будет выводиться перевод описаний на русский язык вместе с оригиналом (английский внизу курсивом).
Без флага `--translate` документация содержит только оригинальные описания на английском языке.

---

## Требования

- Python 3.7+
- Библиотеки:
  - `pyyaml`
  - `deep-translator`

Установить можно так:
```sh
pip install pyyaml deep-translator
```

---

## Использование

```sh
python script.py <шаблон.yaml> [output.md] [--translate]
```

- `<шаблон.yaml>` — путь к YAML-шаблону (экспорт из Zabbix).
- `[output.md]` — имя выходного markdown-файла (по умолчанию `README_template.md`).
- `--translate` — если указан, в документе будут и русский перевод, и оригинал описаний.

---

## Примеры запуска

- Генерация только с английскими описаниями:
  ```
  python script.py zbx_export_templates.yaml
  ```
- Генерация с переводом описаний на русский (и оригиналом внизу):
  ```
  python script.py zbx_export_templates.yaml output.md --translate
  ```
- Помощь по аргументам:
  ```
  python script.py -h
  ```

---

## Что будет на выходе

В результирующем markdown-файле содержатся:
- Название шаблона и его описание
- Таблицы с:
  - Items (метрики)
  - Triggers (триггеры)
  - Macros (макросы)
  - Discovery rules и их прототипами
- В каждой ячейке `Description`:
    - если флаг `--translate` не указан: только оригинал
    - если флаг есть: перевод на русском, ниже *оригинал курсивом*

---

## Особенности

- Перевод осуществляется через Google Translate (deep-translator).
- Если сервис недоступен, будет выведено только оригинальное описание.
- Перевод делается только для полей `description` (описания элементов, триггеров, LLD и т.д.).
- Формат вывода в markdown максимально приближен к документации официальных шаблонов Zabbix.

## Часто задаваемые вопросы

**Можно ли поменять язык перевода?**
Да! В функции `dual_description` поменяйте `target='ru'` на любой нужный язык (например, `'uk'` для украинского).

**Можно выводить оригинал и перевод в две разные колонки?**
Для Excel — легко! Для markdown лучше использовать блок с переводом + оригинал, как реализовано.

**Можно ли добавить свои секции/поля?**
Да, скрипт легко дорабатывается — просто добавьте нужные поля в соответствующие функции.
